# Ruslex — القاموس الروسي الشامل

تطبيق ويب تفاعلي لتعلم اللغة الروسية باللغة العربية.

## الميزات
- قاموس روسي شامل (+25,000 كلمة)
- نظام تكرار ذكي FSRS
- تصنيفات المفردات
- نظام الإعراب (6 حالات)
- أفعال الحركة مع البادئات
- نطق الكلمات (Text-to-Speech)
- المفضلة
- دعم 3 لغات (عربي، إنجليزي، روسي)

## تشغيل المشروع

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```
